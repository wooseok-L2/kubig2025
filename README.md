## Atmega128 - embedded programming

---

- 수업 목표
  - Atmega128을 이용한 임베디드 프로그래밍을 통해 하드웨어와 소프트웨어를 연결하는 방법을 익힌다.
  - Atmega128의 기본적인 사용법을 익히고 이를 이용한 프로젝트를 수행한다.
  - Atmega128의 입출력을 이용한 프로젝트를 수행한다.
